### 1. JS常用注释格式
#### 函数注释格式
```js
/**
 * 函数描述
 * @class 表明改函数为构造函数或类
 * @param {参数类型} 参数名 - 描述信息
 * @return {返回类型} - 描述信息
 * @author 作者信息、日期、邮箱等
 * @version 代码版本号
 */

// 使用示例
/**
 * 两数求和
 * @param {number} x - 参数1
 * @param {number} y - 参数2
 * @return {number} 
 * @author snail-lu 2022/05/01
 * @version 1.0.0
 */
function sum (x, y) {
    return x + y;
}

```
